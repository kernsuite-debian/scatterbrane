Examples
========

To help you get started, we have included some python examples that demonstrate some
of the capabilities of scatterbrane.  The full python scripts should be located in 
the **docs/_code** directory.

.. toctree::

    cp_errors
    ensemble_average
    time_variability

If you have a particular application for which you would like an example, submit a 
pull request to the git repository.  Depending on the interest, future examples could 
include multiple screens, image wander, closure amplitudes, and polarization.
